# Han
>a mobile app to help find local businesses based on user reviews

## Framework
>React, React-Redux, React-Router, LESS

## Build Setup
``` bash

# install dependencies
npm install

# mock server
npm run mock
run at localhost:3000/api

# serve with hot reload at localhost:8080
npm start
```
